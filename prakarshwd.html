Prakarsh 14

Every year Sardar Vallabhbhai Institute of Technology organize PRAKARSH, a national level technical symposium to bring together the best brains in the country and give them a chance to showcase their skills and talents. This would give a platform for the students to interact and compete with each other. There are non-technical events too to incorporate fun activities alongside the technical fervour followed by Pro-nights. In a way,these events helped greatly to develop the potential of budding techno managers from various regions.

Project

1) Prakarsh-14 website

This website had information related to all Technical and Non-Technical events which were organized. Technical Events include Coding competitions,Robotics, Structure and Machine development competitions, and Various Workshops. Where as Non-Technical events include computer gaming competitions,mystery finding games and outdoor games. Moreover, it had registration process for all the events.
Designed front end system using CSS and BootStrap, backend using .NET and database using SQL.

Procedure:

a) Fill an online form.
b) Select all the events in which you are interested.
c) After submitting the form, a confirmation email is sent on the participant's email address with all the events registered information with date and time of specific event.
